# Feature Map

Mapping of key business features to files/components. *(To be fleshed out as development continues)*

- **User Authentication**:
  - Frontend: Firebase integration / Login forms.
  - Backend: `auth` routes, `User` mongoose model.
- **Product Management**:
  - Seller Admin: Product upload and listing forms.
  - Customer Frontend: Product catalog, details page.
- **Order Processing**:
  - Customer Frontend: Cart, checkout, payment integration (Razorpay).
  - Backends: Order creation, status updates, webhook listeners.
- **Real-time Notifications**:
  - Socket.io implementations in both frontends and backends.
